# 🌐 Website Dokumentasi CRUD Laravel + FluxUI

Ini adalah proyek latihan untuk **Pemrograman Web 2** yang berisi dokumentasi dan demo aplikasi CRUD (Create, Read, Update, Delete) menggunakan framework **Laravel** dan library **FluxUI**.

🔗 **Demo Website**: [evanalambana244.github.io/web](https://evanalambana244.github.io/web/)

---

## ✨ Fitur Utama

✅ CRUD Produk (Tambah, Edit, Hapus, Lihat)  
✅ Manajemen Data Pelanggan & Pesanan  
✅ Desain Responsive dengan FluxUI  
✅ Migrasi Database Laravel  
✅ Struktur Kode MVC yang Rapi  

---

## 🛠 Teknologi yang Digunakan

- Laravel 10.x
- FluxUI
- Blade Templating
- MySQL
- HTML, CSS, JavaScript
